
# Crypto currency exchange api

## Description

Api for fast Crypto currency exchange with accounts and balance

## Installation
 - create Postgres user, database
 - create .env file like .env.example, fill all need params

```bash
$ yarn install
$ yarn migrate
$ yarn seed
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
